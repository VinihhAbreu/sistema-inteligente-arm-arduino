# Sistema Inteligente de Irrigação com Arduino e ARM

## Descrição
Projeto desenvolvido para a disciplina de Sistemas Embarcados utilizando Arduino e conceitos introdutórios de Inteligência Artificial.

## Objetivo
Automatizar o controle de irrigação utilizando sensores e algoritmos embarcados.

## Tecnologias Utilizadas
- Arduino UNO
- Linguagem C/C++
- Arquitetura ARM
- Sensor de Umidade
- Relé
- LEDs
- GitHub

## Funcionamento
O sistema monitora a umidade do solo e aciona automaticamente uma bomba de água quando necessário.

## Componentes
- Arduino UNO
- Sensor de Umidade
- Relé 5V
- LEDs
- Protoboard
